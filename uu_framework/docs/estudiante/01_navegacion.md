# Navegación

Cómo moverte por el sitio del curso.

## Barra Lateral

La barra lateral izquierda muestra todo el contenido del curso organizado jerárquicamente.

### Estructura

```
Curso
├── Tareas          ← Acceso rápido
├── Exámenes        ← Acceso rápido
├── Proyectos       ← Acceso rápido
│
├── 1 Introducción  ← Capítulos numerados
│   ├── 1.1 Conceptos
│   └── 1.2 Práctica
├── 2 Git y GitHub
└── A Stack         ← Apéndices con letra
    ├── A.1 Setup
    └── A.2 LLMs
```

### Expandir/Colapsar

- Haz clic en la flecha `>` para expandir una sección
- Haz clic de nuevo para colapsar

### Página Actual

- **Azul/Verde** = Página actual
- **Gris claro** = Sección padre (ancestro)

---

## Migas de Pan (Breadcrumbs)

En la parte superior de cada página verás la ruta de navegación:

```
Inicio > A Stack > 01 Introduction > Cuentas
```

Haz clic en cualquier parte para ir a esa sección.

---

## Navegación Prev/Next

### Parte Superior

Flechas `←` y `→` en la esquina superior derecha.

### Parte Inferior

Tarjetas grandes con el título de la página anterior y siguiente.

---

## Menú en Móvil

En dispositivos móviles:

1. Toca el icono de menú (☰) en la esquina superior izquierda
2. El menú se desliza desde la izquierda
3. Toca fuera del menú para cerrarlo

---

## Secciones Especiales

### Tareas

Lista de todas las tareas del curso con:
- Nombre de la tarea
- Fecha límite
- Estado (pendiente/vencida)

### Exámenes

Información de exámenes:
- Fecha y hora
- Ubicación
- Temas a estudiar

### Proyectos

Proyectos del curso con:
- Descripción
- Fecha de entrega
- Tamaño del equipo

---

## Buscar Contenido

Actualmente no hay búsqueda integrada. Para encontrar algo:

1. Usa la barra lateral para navegar
2. O usa Ctrl+F en tu navegador para buscar en la página actual

---

## Atajos de Teclado

| Tecla | Acción |
|-------|--------|
| `Esc` | Cerrar menú móvil |
| `Ctrl+F` | Buscar en la página |

---

## Consejos

1. **Marca la página principal** en tus favoritos
2. **Revisa Tareas regularmente** para no perder fechas límite
3. **Usa las migas de pan** para volver a secciones anteriores
4. **Explora los apéndices** (A, B) para material de referencia
